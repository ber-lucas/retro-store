Registro:
	1- Criar rota 'post' 
	2- Fazer form de Registro
	3- Criar uma função que compare o e-mail com todos os e-mail presente no banco de dados
	4- Retornar erro se o e-mail já existir
	5- Criar usuário se o e-mail não existir.

Login:
	1- Criar rota 'get'
	2- Fazer o form de Login
	3- Criar uma função que compare o e-mail e senha com todos os dados presente no banco de dados
	4- Retornar erro se o login não existir no banco de dados
	5- Retornar o id do usuário caso esse login exista no banco de dados
	6- Redirecionar o usuário para a página de Store
	7- Guardar o id do usuário para utilizar em outras rotas daí pra frente
